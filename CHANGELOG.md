# Changelog

## 3.0.0 — Causal Case Edition

- Rebuilt every case around one mission, deadline, stakes, and case-specific Saboteur motive.
- Replaced unrelated crises with three dependent phases: trace, secure, and resolve.
- Added rational pre-vote pitches, causal consequences, concrete clues, case progress, and case risk.
- Added a persistent **Case So Far** trail so prior decisions visibly shape later rounds.
- Replaced arbitrary chaos targets with plausible shortcuts that protect the Saboteur by damaging evidence or mission integrity.
- Renamed the social roles to **Investigation Team** and **Inside Saboteur**.
- Expanded schema tests to enforce one careful plan, one tradeoff, and one unique high-risk target per phase.
- Preserved fully credential-free, zero-runtime-API gameplay.

## 2.0.0 — Plot Saboteur Edition

- Replaced the improv-performance loop with an 8–12 minute social deduction game.
- Added one hidden Plot Saboteur with three private target outcomes.
- Added anonymous pass-the-phone ballots, public vote splits, and 30-second discussion rounds.
- Added final secret accusations and a clear two-condition win rule.
- Added a verdict screen with accusation counts, target successes, chaos score, and a three-round damage ledger.
- Reworked all four bilingual story packs around defensible choices and escalating consequences.
- Added duplicate-name protection and deterministic integrity tests.
- Preserved fully credential-free, zero-runtime-API gameplay.

## 1.0.0 — Delivery Edition

- Added complete Simplified Chinese and English gameplay.
- Made English the default experience with one-tap switching to Chinese.
- Added **Moon Motel**, a fourth original story world.
- Added a first-run rules panel and per-act 60-second performance timer.
- Added seeded role rotation, choice remixing, and four local callback variants.
- Added a three-act finale recap with player attribution.
- Improved mobile controls, focus states, responsive layouts, and reduced-motion behavior.
- Kept the deployed experience fully zero-API and credential-free.
